# CRUDS - Product Management System

A lightweight, local-storage-based Product Management System (CRUDS) built with vanilla HTML, CSS, and JavaScript. 

[👉 Click here to view the live app](https://fergo00.github.io/cruds-product-management/)

## 🚀 Features
- **Create**: Add new products with automatically calculated total prices (price + taxes + ads - discount).
- **Read**: View all added products in an organized, clean table.
- **Update**: Edit details of existing products dynamically.
- **Delete**: Remove individual products or clear the entire inventory at once.
- **Search**: Fast search capabilities filtering products by Title or Category.
- **Persistent Data**: Powered by browser LocalStorage to save your data even after refreshing or closing the page.
- **Smart UX**: Seamlessly cycle through inputs and submit items using the `Enter` key.

## 🛠️ Technologies Used
- HTML5
- CSS3 (Responsive Design & Transforms)
- JavaScript (Vanilla ES6, LocalStorage API, DOM Manipulation)
